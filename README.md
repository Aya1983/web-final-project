# منصة الدعم النفسي | Mental Support Platform

## Problem - Solution
### المشكلة:
الأزمات والحروب تترك آثاراً نفسية عميقة على الأفراد، خاصة الأطفال والشباب. كثيرون يحتاجون للدعم النفسي، لكن الوصول إلى خدمات موثوقة وسريعة أمر صعب، ويواجه البعض حاجزاً نفسياً لطلب المساعدة.

### الحل:
تقدم منصة الدعم النفسي مساحة آمنة لتلقي الدعم النفسي والاجتماعي، من خلال:
- جلسات استماع فردية ومجتمعية
- استشارات مع متخصصين
- موارد ومقالات تثقيفية
- برامج دعم للأطفال
- فرص تطوع للمساهمة في المجتمع



## المستخدم المستهدف
- الأفراد المتأثرين بالحروب والأزمات.
- الأطفال والشباب الذين يحتاجون للدعم النفسي.
- أي شخص يرغب بفهم الصحة النفسية والتعامل مع الصدمات.
- المجتمع الراغب بالمشاركة في تقديم الدعم التطوعي.



## أهم الميزات
1. دعم نفسي أولي: جلسات استماع لتقييم الحالة وتقديم الدعم الفوري.
2. مجموعات دعم: جلسات جماعية لمشاركة التجارب والتعافي سوياً.
3. تثقيف نفسي: مقالات وموارد لفهم الصدمات وآليات التعامل معها.
4. رعاية الأطفال: برامج مخصصة للأطفال المتأثرين بالأزمات.
5. استشارات متخصصة: ربط مع مختصين ومعالجين نفسيين مرخصين.
6. برامج تطوعية: فرص للمشاركة المجتمعية وتقديم الدعم للآخرين.
7. محتوى متعدد الوسائط: مقالات، حملات توعوية، وفيديوهات دعم نفسي.
8. نموذج حجز مباشر: تواصل سريع مع المنصة وطلب جلسة أو استشارة.


## التقنيات المستخدمة
- HTML5 & CSS3 & JavaScript: لبناء واجهة المستخدم.
- Bootstrap 5: لتصميم واجهة متجاوبة وسهلة الاستخدام.
- jQuery: لإضافة التفاعلية وتصفية المحتوى وإدارة المودال.
- Font Awesome: أيقونات تعبيرية وجذابة.
- Custom CSS & Animations: تأثيرات انتقالية وتفاعل سلس.
- Video & Media Integration: دعم الفيديوهات التعليمية والتحفيزية.
- Responsive Design: دعم جميع الأجهزة والشاشات.
- Form Validation & Feedback: تجربة مستخدم سلسة لنماذج التواصل.


## طريقة الاستخدام
1. زيارة الصفحة الرئيسية للتعرف على المنصة وخدماتها.
2. استعراض الخدمات والمحتوى التعليمي.
3. حجز جلسة دعم نفسي أو استشارة عبر نموذج التواصل.
4. متابعة حملات الدعم والمشاركة في المبادرات التطوعية.
5. الاستفادة من محتوى الوسائط والفيديوهات التوعوية.
